"Tiny Box" 2x2 Crossnumber Puzzles

All these puzzles use the same grid and labels
  A ↓
X → 		B ↓  
Y →  

(t) = solution only unique up to transposing - i.e. for these puzzles you could swap A,B with X,Y and it still works

x|y (or "x divides y") means the same as "x is a factor of y", or "y is a multiple of x", meaning that y is equal to x multiplied by some integer (whole number).

A number is called a palindrome if it is the same when written backwards. For instance, 70307.

  1. X = 5Y
    AB is a square
  2. X = 5Y
    (A - B) is a square
  3. X, Y and B are distinct squares
  4. X|B and A|Y
    X < A
  5. (t) X, Y, A, B are all the same number but
    written in four different bases, all < 10
  6. (t) X, Y, A, B are all the same PRIME number but
    written in four different bases, all ≤ 10
  7. (A + X)2 is a palindrome
    B2 - Y2 is a square
  8. (A + X) = 4(B + Y)
    B > Y
  9. Y = 2B
    A = 3X - 44
  10. A = 2B
    Y = 4X
